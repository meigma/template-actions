# Security Policy

## Supported versions

Only the latest released version receives security fixes.

## Reporting a vulnerability

Report vulnerabilities through
[GitHub private vulnerability reporting](https://github.com/meigma/template-actions/security/advisories/new).

Do not use public issues, pull requests, discussions, or chat channels for
vulnerability reports. Include the affected version or commit, impact,
reproduction steps, relevant logs, and any suggested mitigation when available.
